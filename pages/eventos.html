Eventos click

Click

nav.addEventListener('click', ()=> {
console.log('Hello World')
})

Mouseenter

Mouse sobre un elemento.

nav.addEventListener('mouseenter', ()=> {
console.log('Hello World')
})

Mouseout

Mouse deja de estar sobre un elemento.

nav.addEventListener('mouseout', ()=> {
console.log('Hello World')
})

Mousedown

Se ejecuta cuando mantenemos el click en un elemento.

nav.addEventListener('mousedown', ()=> {
console.log('Hello World')
})

Mouseup

Se ejecuta levantamos el click.

nav.addEventListener('mouseup', ()=> {
console.log('Hello World')
})

Dblclick

Se ejecuta cuando ahcemos doble click.

nav.addEventListener('dblclick', ()=> {
console.log('Hello World')
})

Eventos con el teclado

Keydown

Se ejecuta cuando precionas una tecla.

input.addEventListener('Keydown', () => {
console.log('Escribiendo..')
})
  • keyup = soltar tecla.
  • blur = click al input.
  • copy = Se ejecuta cuando haemos Ctrl + c.
  • paste = Se ejecuta cuando haemos Ctrl + v.
  • cut = Se ejecuta cortamos Ctrl + x.
  • input = Cumple las mismas funciones que las anteriores, exepto el blur.

Target

input.addEventListener('input', (e) => {
console.log(e.target.value)
})

Eventos que suceden en un Formulario

Submit

Sucede cuando ejecutamos submit a un Formulario.

e.preventDefault resetea la ejecucion por defecto de recargar el sitio web

input.addEventListener('submit', (e) => {
e.preventDefault()
console.log()
})

Eventos que seceden con el scroll

scroll

Detectando la cantidad de scroll en vertical basado en pixeles.

window.addEventListener('scroll', () => {
const scrollPX = window.scrollY;
console.log(scrollPX)
})

scroll and getBoundingClientRect()

Detectando la cantidad de scroll en vertical basado en pixeles.

window.addEventListener('scroll', () => {
const section = document.queryselector('.section')
const ubication = section.getboundingClientRect();
if(ubication.top < 784){
console.log('El elemento ya esta visible.')
}else{
console.log('Aun no, da mas scroll.')
}
})

Event Bubbling

stopPropagation

Detiene la ejecucion de eventos tipo burbuja

Esto sucede cuando hacemos click en un elemento hijo que tiene un evento click y su padre tambien, ejecutanto asi los dos eventos click.

para evitar la propagacion de los eventos utilizamos stopPropagation.

cardDiv.addEventListener('click', ()=> {
e.stopPropagation();
console.log('click en div');
})

infoDiv.addEventListener('click', ()=> {
e.stopPropagation();
console.log('click en div');
})

tituloDiv.addEventListener('click', ()=> {
e.stopPropagation();
console.log('click en div');
})

if

Otra manera para el stop propagation es con if/ condicionales

ejemplo

cardDiv.addEventListener('click', e => {
if(e.target.classList.contains('titulo')){
console.log('diste click en titulo')}
})

cardDiv.addEventListener('click', e => {
if(e.target.classList.contains('precio')){
console.log('diste click en precio')}
})

cardDiv.addEventListener('click', e => {
if(e.target.classList.contains('card')){
console.log('diste click en card')}
})
